# anpr-ocr
Automatic Number Plate Recognition using OCR

This script was modified to work with license plates used in Mexico. 

# Project Structure

```
|-- anpr-ocr/
|   |-- anpr/
|   |   |-- anpr.py
|   |-- license_plates
|   |-- ocr_license_plate.py
```
